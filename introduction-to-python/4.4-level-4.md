# :material-numeric-4-circle: Assessment Level 4

Level 4 checks that you can combine inputs, variables, number conversion, arithmetic and selection to solve a more challenging problem.

---

## :material-ticket-confirmation-outline: Task: Cinema Ticket Price

Create a program that:

- Asks the user to type in their age.
- Asks the user to type in the normal ticket price.
- Converts the age into an integer.
- Converts the ticket price into a decimal number using `float()`.
- Gives a child ticket if the age is less than `16`.
- Displays the correct ticket price.

A child ticket costs half of the normal ticket price.

Example:

```text
Enter your age: 12
Enter the normal ticket price: 8
Your ticket price is 4.0
```

---

## :material-check-circle-outline: Your Program Must

- Use two separate inputs.
- Store each answer in a variable.
- Convert the age using `int()`.
- Convert the ticket price using `float()`.
- Use selection to check the age.
- Use arithmetic to calculate the child ticket price.
- Display the correct ticket price.
- Work with different ages and ticket prices.

---

Test your program with at least three different ages and ticket prices.
