# :material-clipboard-check-outline: Python Programming Assessment Rubric

Use this rubric in Google Classroom by adding **one criterion for each assessment level**.

Each level is worth **10 marks**.

In Google Classroom, the boxes at the bottom of each criterion are **performance levels**. Students are assessed against these performance levels.

For each criterion, add these four performance levels:

| Points | Level Title | Description |
|-----------|-----------|-----------|
| 1 | Beginning | Shows limited evidence of the skills for this level. |
| 4 | Developing | Shows some evidence of the skills for this level. |
| 7 | Progressing | Shows clear evidence of most skills for this level. |
| 10 | Secure | Shows clear and accurate evidence of all skills for this level. |

This gives one selectable mark for each level.

---

## :material-google-classroom: How to Enter This in Google Classroom

Create **4 criteria**:

| Criterion Title | Criterion Description | Performance Levels |
|-----------|-----------|-----------|
| Level 1 | Reading, correcting and explaining simple Python code. | `1` Beginning, `4` Developing, `7` Progressing, `10` Secure |
| Level 2 | Create short programs with inputs, variables, outputs and simple selection. | `1` Beginning, `4` Developing, `7` Progressing, `10` Secure |
| Level 3 | Create programs that use number input, conversion and selection to compare values. | `1` Beginning, `4` Developing, `7` Progressing, `10` Secure |
| Level 4 | Create programs that combine inputs, variables, conversion, arithmetic, selection and testing. | `1` Beginning, `4` Developing, `7` Progressing, `10` Secure |

Your rubric should show **/40** when all four criteria have a maximum score of `10`.
