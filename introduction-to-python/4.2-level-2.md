# :material-numeric-2-circle: Assessment Level 2

Level 2 checks that you can use inputs, variables and outputs in short programs.

---

## :material-movie-open-outline: Task 1: Favourite Film

Create a program that:

- Asks the user to type in their favourite film.
- Stores the answer in a variable.
- Displays the film name back to the user in a sentence.

Your output should include:

```text
I'd give it 4 stars
```

Example:

```text
I watched Inside Out and I'd give it 4 stars.
```

---

## :material-help-circle-outline: Task 2: Explain the Code

Look at this line of code:

```python
item = input("What is your favourite film? ")
```

Explain what the line does.

Your answer should mention:

- The question shown to the user.
- The user's answer.
- The variable used to store the answer.

## :material-emoticon-happy-outline: Task 3: Happy Checker

Create a program that:

- Asks the user if they are happy.
- Tells the user to answer `yes` or `no`.
- Stores their answer in a variable.

If the user enters `yes`, display:

```text
Glad you are happy!
```

Otherwise, display:

```text
What's up?
```
