# :material-numeric-3-circle: Assessment Level 3

Level 3 checks that you can use selection with numbers.

---

## :material-greater-than: Task 1: Largest Number

Create a program that:

- Asks the user to type in the first number.
- Asks the user to type in the second number.
- Stores both numbers in variables.
- Decides which number is larger.
- Displays the largest number on the screen.

Remember to convert both inputs into integers.

Example:

```text
Enter the first number: 12
Enter the second number: 7
The largest number is 12
```

---

## :material-school-outline: Task 2: Test Score

Create a program that:

- Asks the user to type in their test score out of `100`.
- Stores the score in a variable.
- Displays `YOU PASSED!!` if the score is `50` or greater.
- Displays `YOU FAILED!!` if the score is less than `50`.

Example:

```text
Enter your score: 62
YOU PASSED!!
```

## :material-lightbulb-on: Hints

- Use `int(input(...))` for number inputs.
- Use `>` or `>=` to compare numbers.
