# Comparison Operators
In IF statements we will be looking to compare some data.
We might want to see if a variable is equal to a number or a piece of text.
We might want to see if a variable contains a number greater than another
Or contains a number smaller than another.

In python we use certain symbols to compare data.
To see if something is mathematically equal to another we use a double equals sign (==)
To see if a variable contains a number greater than another we use the greater than symbol (>)
To see if a variable contains a number smaller than another we use the less than symbol (<)
