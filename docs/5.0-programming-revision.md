# :material-refresh: Python Programming Revision

These revision questions practise the skills from the Python basics unit.

You should only use:

- `print()`
- `input()`
- Variables
- Strings, integers and floats
- `int()` and `float()`
- Arithmetic operators: `+`, `-`, `*`, `/`
- Comments
- `if` and `else`
- Comparison operators: `>`, `<`, `==`, `>=`, `<=`, `!=`

!!! warning "Important"

    Do not use loops, lists, functions, `elif` or any techniques that have not been taught yet.

---

## :material-tools: Revision Rules

For every program:

- Use clear variable names.
- Ask the user for input where needed.
- Convert numbers using `int()` or `float()`.
- Display outputs with helpful messages.
- Use comments to explain important parts of your code.
- Test your program with more than one set of inputs.

---

## :material-format-list-checks: Revision Questions

!!! question "Complete the revision questions below."

    **Task 1: About Me Card**

    Ask the user for:

    - Their first name
    - Their favourite school subject
    - Their favourite hobby

    Display the answers as a short profile.

    **Example output:** `Sam enjoys Art and likes skateboarding.`

    ---

    **Task 2: Pet Name Tag**

    Ask the user for:

    - A pet's name
    - The type of animal
    - The owner's name

    Display a name tag message.

    **Example output:** `Milo is a cat and belongs to Aisha.`

    ---

    **Task 3: Book Page Calculator**

    Ask the user for:

    - The number of pages they read each day
    - The number of days they read for

    Calculate and display the total number of pages read.

    **Formula:** `total_pages = pages_per_day * days`

    ---

    **Task 4: Water Bottle Tracker**

    Ask the user for:

    - The size of one water bottle in litres
    - How many bottles they drank

    Calculate and display the total litres of water they drank.

    **Formula:** `total_litres = bottle_size * number_of_bottles`

    ---

    **Task 5: Homework Time**

    Ask the user for:

    - The number of minutes spent on Maths homework
    - The number of minutes spent on English homework
    - The number of minutes spent on Computing homework

    Calculate and display:

    - The total homework time
    - The average time per subject

    **Formula:** `average = total / 3`

    ---

    **Task 6: Locker Height Checker**

    Ask the user for:

    - Their name
    - The height of their bag in centimetres

    If the bag height is less than or equal to `45`, display a message saying it will fit in the locker.

    Otherwise, display a message saying it is too tall for the locker.

    Your message should include the user's name.

    ---

    **Task 7: Quiz Point Checker**

    Ask the user for:

    - Their team name
    - Their quiz points

    If the points are greater than or equal to `20`, display a message saying the team has reached the target.

    Otherwise, display a message saying the team needs more points.

    ---

    **Task 8: Secret Word Check**

    Ask the user to enter a secret word.

    If the word is equal to `revision`, display:

    ```text
    Correct secret word.
    ```

    Otherwise, display:

    ```text
    Incorrect secret word.
    ```

    ---

    **Task 9: Train Fare Calculator**

    Ask the user for:

    - Their name
    - Their age
    - How many train journeys they need

    One train journey costs `2.80`.

    Calculate the normal total cost.

    If the user is under `16`, they pay half price.

    Otherwise, they pay the normal total cost.

    Display:

    - The user's name
    - Number of journeys
    - Normal total cost
    - Final amount to pay

    **Hint:** `half_price = normal_total / 2`

    ---

    **Task 10: Plant Growth Check**

    Ask the user for:

    - The plant name
    - Its starting height in centimetres
    - Its new height in centimetres

    Calculate how much the plant has grown.

    If the plant has grown more than `5` centimetres, display a message saying it has grown well.

    Otherwise, display a message saying it needs more time.

    Display the amount grown in your output.

    ---

    **Task 11: School Trip Lunch Order**

    Ask the user for:

    - The student's name
    - Number of sandwiches
    - Number of fruit bags
    - Number of juice cartons

    Use these prices:

    | Item | Price |
    |-----------|-----------|
    | Sandwich | `1.80` |
    | Fruit bag | `0.90` |
    | Juice carton | `0.75` |

    Calculate and display:

    - Sandwich total
    - Fruit total
    - Juice total
    - Final lunch cost

    If the final lunch cost is greater than `5.00`, display:

    ```text
    Check this order with a teacher.
    ```

    Otherwise, display:

    ```text
    Order accepted.
    ```

    ---

    **Task 12: Revision Session Planner**

    Ask the user for:

    - Their name
    - Minutes spent revising Python
    - Minutes spent revising Maths
    - Minutes spent revising English

    Calculate and display:

    - Total revision time
    - Average revision time per subject

    If the total revision time is greater than or equal to `60`, display a message saying they met the revision target.

    Otherwise, display a message saying they should plan another short session.

    Your final output should include the user's name.

---

## :material-checkbox-marked-circle-outline: Before You Finish

Check that each program:

- Runs without errors.
- Uses `int()` or `float()` for number inputs.
- Uses `if` and `else` correctly where needed.
- Has correct indentation after `if` and `else`.
- Gives clear output messages.
