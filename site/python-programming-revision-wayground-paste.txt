Python Programming Revision MCQ

1. Which Python command is used to display an output on the screen?
A. input()
B. print()
C. int()
D. float()
Answer: B

2. What is an output in a program?
A. Information displayed by the computer
B. Information typed by the user
C. A named storage place
D. A type of calculation
Answer: A

3. What does input() do in Python?
A. Displays a message only
B. Stops the program forever
C. Asks the user to enter information
D. Converts text into a number
Answer: C
